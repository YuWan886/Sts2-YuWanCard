## 安装

1. 下载最新版本[Release](https://github.com/YuWan886/Sts2-YuWanCard/releases/latest)
2. 下载前置依赖mod[Baselib](https://github.com/Alchyr/BaseLib-StS2)
3. 解压到游戏目录下的`mods`文件夹中
